Conforme definido no [CODEOWNERS](../.github/CODEOWNERS), alterações neste diretório requerem aprovação de:
- @luan0x73
- @payment-sec

## Segurança

⚠️ **ALTA CRITICIDADE** - PCI-DSS Compliance Required

Este é o diretório mais sensível do repositório. Todas as mudanças passam por:
- Review obrigatório de 2 aprovadores (incluindo security lead)
- Scan de secrets com padrões adicionais para dados de cartão
- Análise de vulnerabilidades com threshold CRITICAL
- Auditoria completa de mudanças
